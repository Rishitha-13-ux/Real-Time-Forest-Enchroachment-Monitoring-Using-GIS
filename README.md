# Real-Time-Forest-Enchroachment-Monitoring-Using-GIS

This project implements a real-time forest fire classification system using CNN and RNN + LSTM.

## Features
- Image classification (Fire / Non-Fire)
- Real-time webcam detection
- Prediction score and confidence display

## Technologies Used
- Python
- TensorFlow / Keras
- OpenCV

## Files Included
- fire_classification.ipynb
- fire_model.h5
- Final_Report.pdf
